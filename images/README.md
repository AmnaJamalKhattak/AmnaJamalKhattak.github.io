# Profile Image

Please add your profile photo here and name it `profile.jpg`.

**Instructions:**
1. Add your professional photo to this folder
2. Name it `profile.jpg` (or update the filename in `index.html` if you use a different name)
3. Recommended size: 400x400 pixels or larger (square aspect ratio works best)
4. The image will be automatically displayed as a circular avatar

**Supported formats:** JPG, PNG, WebP

If no image is found, a placeholder icon will be displayed instead.
